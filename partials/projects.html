Projects

I-ALiRT

Real-time telemetry ingest, processing, and distribution from international ground stations. I-ALiRT (IMAP Low-latency Alert and Real-Time Telemetry) provides rapid access to IMAP science data for space weather monitoring and response.

IMAP Data Systems

AWS CDK infrastructure, APIs, and science data workflows supporting mission operations for the Interstellar Mapping and Acceleration Probe (IMAP). Includes automated pipelines for ingesting, processing, and distributing Level 0 through Level 3 science data products.

SPICE & Geometry

Pointing frames, coordinate transforms, and data processing for heliophysics pipelines using SPICE kernels. Enables accurate geometric calculations for instrument pointing and data calibration across mission phases.
